Basic website setup for SEO

To start a website SEO make sure these seven steps to successful SEO.
1.Make sure that your website is indexed.
2.Set up GSC and Bing Webmaster tools.
3.Check the Search Console for manual actions.
4.Set up Google Analytics.
5.Create a Robots.txt file.
6.Install and configure an SEO plugin.
7.Generate & submit a sitemap.
